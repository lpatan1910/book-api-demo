# BOOK API - Spring Boot, JPA Rest API

Restful CRUD API for a simple Book-Manage application

## Requirements

1. Java - 1.8.x

2. Gradle - 3.3.9

3. H2

## Rest APIs

The app defines following CRUD APIs.

    GET /api/v1/books - to list all 
    
    POST /api/books
    
    GET /api/books/{bookId}
